unidades de medida


Dentro do CSS existem as absolutas e as relativas.



Absolutas

As unidades de media absolutas são as que não se necessita de nada para serem interpretadas,um metro não vai deixar de ser um metro por estar em outra unidade de medida , pode estar com uma unidade e valor diferente mas seu diamêtro não vai deixar de ser o mesmo de que a de um metro.

Como já sabemos , é com CSS que fica a tarefa das estética da página.Suas própriedades sempre vão necessitar de uma unidade de media para funcionar e elas podem ser :


Centímetros(cm)

milímitros(mm)

polegadas(in)

obs:1in=96px=2.54cm

pixels(px)

pontos(pt)

pica(pc)

obs:1pc=12pt



Unidades de medida relativas





As unidades de medida relativas funcionam de uma maneira com que se possa acompanhar quaisquer alterações com o intuito de preservar suas dimensões primárias.

Imaginemos uma situação onde exista uma "DIV pai" e um "DIV filho".Dentro do que diz respeito as unidades de medida relativas , toda vez que se altera a DIV pai ,vai se alterar tambem a DIV filho .

Uma unidade das mais famosas dentro das relativas é a porcentagem(%),podendo ser aplicada tanto no que diz repeito a altura do que para largura.


View Port

Em computação gráfica, a viewport é a porção de área visível de um plano ou , em outras palavras e de uma maneira mais simples , seria a sua tela .

A View Port tambem é uma unidade de medida relativa onde ,quando utilizada , mantem as dimensões mesmo diante das alterações feitas na tela , caso se dê um zoom ou se distâncie.
