Introduçâo

O HTML foi criado em 1991 para suprir uma necessidade que os cientistas tinham para compartilhar documentos de forma fácil através da internet entre as instituiçôes de ensino.

Existia a dificuldade de se enviar documentos e arquivos pois o computador não conseguia distinguir títulos , parágrafos,pontos continuativos e finais .

As TAGS servem para ajudar o PC a entender melhor as linhas e facilitar sua vida.Ná prática,na hora da programaçâo,a maneira que se usa as TAGs é usando os sinais < (maior quê) e > (menor quê) e ,entre elas,o nome da TAG que se deseja usar

Uma outra necessidade é informar ao computador até onde as TAGs se encerram .Para isso ,existe as TAGs de fechamento,onde a única diferença das TAGs de abertura é a necessidade de por o sinal de barra ( / ) logo após o sinal de menos quê

Elemento html

Elemento html sempre vai ser tudo aquilo que você precisa para representar alguma coisa(parágrafo,títulos,imagens e etc..)

Os elementos sao formados por: < nome da TAG > ( TAG de abertura) conteúdo (TAG de fechamento)

Estrutura básica do HTML

Para se começar a criar a sua primeira página WEB, a primeira coisa que se precisa fazer é dizer ao navegador como ele deve interpretar a página:

<!DOCTYPE HTML>

Isso é a primeira coisa que se faz .Quando se faz isso,você está dizendo para o seu navegador que todo o resto que estiver dentro desta página ,ele precisa interpretar como html 5 ,a versão mais atualizada e é provável que se veja com bastante frequência no mercado

<html> </ html>

Diz onde começa e termina sua página

<head ></ head>

Indicará tudo que seu computador precisa saber para carregar a sua página corretamente.É aqui onde se coloca o título que o naavegador vai indicar na aba da página acessada

<body></ body>

Vai entrar tudo que você precisa mostrar para o usuário dentro da página: textos,imagens,links... , tudo que se precisa mostrar ao usuário dentro da página.


Então, a estrutura básica do html e sua hierarquia é :


1 <!doctype html>

2 <html>

3 <head >

conteúdo

4 </head>

5 <body>

conteúdo

6 </body>

7 </html>


Meta,title,atributos e TAG de auto fechamento


title

É título em inglês,ele é uma TAG filho da HEAD e é nela que usamos para fazer o elemento HTML e nomear a barra da navegaçâo .No caso,iria aparecer um BUG devido o ascento agudo e temos que concertar

exemplo:

<!doctype html>

<html>

<head>

<title> exemplo </ title>

</ head>

<body>

</ body>

</ html>


<meta charset="UFT-8">

O navegador não conseguia interpretar o ´´pá´´ da palavra página devido o ascento agudo e, então , precisamos dizer ao programa em qual alfabeto ele precisa se basear ,já que a nossa língua possue muitos ascentos . Essa TAG em questão é diferento que todos os elementos que se viu antes , onde se tem uma TAG de abertura , conteúdo e de fechamento. Essa TAG se chama self closing TAG ( uma TAG que se fecha em si mesma),ela não tem nenhum conteúdo ela somente serve para dizer ao navegador como ele deve interpretar algo ,no exemplo atual ,como ele deve interpretrar o alfabeto portugues brasileiro.

exemplo:

<!doctype html>

<html>

<head>

<title> exemplo < / title >

<meta charset = ´´ uft-8´´ >

</ head>

<body>

</ body>

</ html >

atributo

Um atributo sempre vai ser tipo uma palavra chave ou algo do gênero ,que sempre vai entar antes do maior quê ( > ) de uma TAG de auto-fechamento.O atributo não é exclusivo para uma TAG de auto fechamento , qualquer TAG pode se usar mas elas podem variar ,é possível que funcione em uma TAG mas não em outra .

charset


É um atríbuto que serve justamente para informar que tipo de caractér ou alfabeto que se utiliza.UFT=8 é o tipo ou formato de alfabeto ou caracter que queremos usar .

Título,parágrafo e semântica

Quando se trata sobre a formação do conteúdo da página , toda a operação se faz dentro do BODY


. HeadLine

Esse é o nome para título em inglês reprensentada pela TAG < h >.É muito comum utilizar a primeira letra da palavra em inglês

. Paragraph

É o nome parágrafo em inglês e se utiliza usando a TAG < p >

. Semântica

(h1,2,3,4,5,6).Essa numeração não é somente referente ao tamanho da TAG (HeadLine),é tambem referente ao grau de impotância que seu navegador vai dar para sua página.
Quando buscamos sobre algo no navegador ,geralmente botamos a palavra chave referênte ao assunto ou tema que se deseja pesquisar
exemplo:

>

No client side , foi pesquisado : jogar futebol

E os resultados seriam...


1 Página 1

Como jogar futball

Alguns paços para se melhorar no futball

,,,,,,,,,,(restante do conteúdo da página),,,,,

2 Página 2

Melhore no futball

Alguns paços para se melhorar no futball

,,,,,,,,,,,(restante do conteúdo da página),,,,,


No exemplo, o navegador iria sugerir a Página 1 ,porque ele se baseia nas TAGs mais importântes (com maior semântica)e as que chegam o mais próximo possivel das palavrs que foram usadas no client side.


H1 É o mais importante

H2 É a segunda mais importante e assim por diante....


Comentário

A TAG do comentário é : < !--o conteúdo que deseja-->
