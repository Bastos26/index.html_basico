Lang e Div

lang

  • Lang é um atributo que ajuda os mecânismos de busca e aos navegadores a entender qual exatamente é o conteúdo da sua Página no sentido de idioma.Serve para identificar para o navegador e o mecanismo de busca qual é a linguagem da página

exemplo...

html Lang="pt-br"

sendo que o PT se remete a língua e o BR refere-se ao país específico

Sua aplicaçâo é na região HEAD

DIV

  • Div é uma TAG que serve para agrupar as linhas e orgânizar o código e elementos

    • E possivel usar os Div´s em links da pópria página com o intuito de deixar o projeto mais dinâmico para o cliente,deixando mais fácil para ele encontrar algo específico dentro do projeto

    exemplo...

    < a href="#exemplo>exemplo< /a >

    o sinal # se necessário para fazer a linkagem dentro do projeto
