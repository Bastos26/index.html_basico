link


Um link é quando você liga uma página em outra facilitando o acesso entre elas.

TAG de âncoragem

(< a>< /a>)

Essa é a TAG que se utiliza na crianção dos links

Atributo HREF

ESte atributo serve como um complemento para a criação do link ,uma configuração para funcionar corretamente


Existem duas maneiras de se adicionar os endereços dos links:

Colocando todo domínio da página em que se deseja(http://www.exemploquequiser.com) Colocando o nome da pasta criada do seu domínio(exemplo em questão que estou utilizando)


Então,a TAG completa seria:

< a href="http://www.exemploque quiser.com ou o nome da página do seu dominio"exemplo do nome do link(é aqui que se denomina o nome do link em que se deseja,sempre antes do fechamento)< /a>
